# Gunjan Negi — UI/UX Designer Portfolio

A simple, single-page portfolio built with plain **HTML, CSS, and JavaScript**.

## Sections
1. **Hero** — quick intro to who Gunjan is
2. **Projects** — MindSpark Education + Digital Agency Landing Page Redesign (Behance)
3. **Tools** — Figma, Jitter, Framer, Canva, VN Editor
4. **About** — design philosophy with LinkedIn & YouTube links
5. **Contact** — email and phone

## Color palette
| Role | Color | Hex |
|------|-------|-----|
| Background (60%) | Warm White | `#FAFAF7` |
| Primary | Soft Teal | `#4ECDC4` |
| Secondary | Warm Peach | `#FFB347` |
| Accent (10%) | Soft Lavender | `#C9B8E8` |

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files
- `index.html` — page structure
- `styles.css` — styling and responsive layout
- `script.js` — mobile menu, sticky nav, scroll reveal
- `assets/gunjan.png` — profile photo
